# Changelog Archive: v0.16.x Series

## [0.16.3] - Unreleased

### Added

- Added `yams graph --explore <query>` for agent-oriented graph context: ranked symbols, canonical relationships, and line-numbered source snippets with output budgets.
- Added graph context service contracts for explore/lookup/trace/impact/affected-tests flows; explore is implemented in-process for CLI use.

### Changed

- Search and grep follow-up hints now prefer `yams graph --explore ...` for source-context workflows.
- Graph relation aliases are normalized through shared metadata helpers so CLI, app services, and daemon dispatcher paths agree on relation names.

### Tests

- Added graph context service tests for ranking, test-file handling, snippet budgets, omitted-code mode, and canonical relationship rendering.
- Added CLI smoke coverage for `yams graph --explore` JSON and human output.
